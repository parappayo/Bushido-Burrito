
Shmup Project

A demo shoot-em-up style game engine written in C.


Build Instructions:

- A project file for Code::Blocks is included. See: shmup.cbp
- The libpng library is required to link. See: http://libpng.org/
- The libpng lib also depends on zlib.
- Compiled libraries should go in ./lib/
- Runtime DLLs should go in ./
- Library headers should go in ./include/
- A sprite graphic is used, copy ../../sprites/plane01.png to the runtime dir.

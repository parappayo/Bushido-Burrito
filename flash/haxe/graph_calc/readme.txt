
About graph_calc
---

This project is a simple graphing calculator written in Haxe targetting Flash.
It includes a rudimentary math formula parser.

Building graph_calc.swf:

	haxe compile.hxml


Running the Smoke Tests
---

First build test.js using the following command:

	haxe test.hxml

Now open test.html.  The smoke tests will run and the result will be displayed.
The focus of the smoke tests is to stress the math parser component of the
project.
